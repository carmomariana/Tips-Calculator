## Tip Calculator

Is an Android app to help you to calculate the amount of tip and split the table bill.

## Techs

- Kotlin
- Activities
- ConstraintLayout
- Navigation with Intents
- Material3 themes and styles
- UI Components
  - TextView
  - EditText
  - Button
- Binding
- Translate EN-US and PT-BR
